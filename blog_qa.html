What is DOM ? And What is purpose of DOM?

DOM Stands for document object model..It's a model where html document as a tree structure wherein each node in an object.Every node representing a part of the document.You can change structure ,style and content using it.The main purpose of dom is accessing document dynamically and add,remove and create element dynamically using dom.

How will you select HTML Element using DOM?Name the DOM Method

I can select html element using dom by using html tag name, class name and id and using querySelector method.. use getElementByTagName(use this for selecting tag) getElementByClassName(selecting element using class name) getElementById( selecting element by using id), querySelector(selecting class and for first name just from same name class),querySelectorAll(for same name all class)

Some Dom Method name: appendChild()->use this for add a new child node for an element
click()->when mouse click on html document
blur()-> used to focus document
onmouseover() -> when over mouse on a specific element
getAttribute()-> display an element attribute value

What is Callback function and why it's use ?

callback is for execute code in response to a specific event.callback function is a function which passed into another function as an argument.The main purpose of using callback function is that it's ensure that the function will not run before the task is finished but it will run after the task or event finished
